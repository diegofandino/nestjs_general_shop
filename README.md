# Nest TESLO SHOP

### This is a general ecommerce template made in Nestjs.

1. Clone project
2. Create .env file based on .env.example
    ```bash
    cp .env .env.example 
    ```

3. Run docker-compose up -d
    ```bash
    docker-compose up -d
    ```
4. Run yarn install
    ```bash
    yarn install
    ```
5. Run seed
    ```bash
    http://localhost:3000/api/seed
    ```

6. Run yarn run start:dev
    ```bash
    yarn run start:dev
    ```
